Busca de voos LATAM — múltiplos destinos (15s entre cada)

Selecione origem, destino(s) (códigos IATA) e a data. O autosuggest usa a base global (CDN) com fallback offline.

Como funciona & pré-requisitos
  1. Faça login na LATAM em outra aba antes de usar (obrigatório).
  2. Autorize pop-ups para este site.
  3. Ao clicar em Buscar, abrimos abas de espera (about:blank) para cada destino:
    • A 1ª aba navega para a LATAM imediatamente.
    • As demais auto-navegam a cada 15s (0s, 15s, 30s, ...), mesmo que você troque de aba, pois o timer roda dentro de cada aba (meta-refresh).
    • O botão Parar fila fecha as abas ainda em espera.
  4. Você pode colar uma lista de IATAs (vírgula, espaço ou quebra de linha).
Clique no campo para ver sugestões. Digite para filtrar.
Use "Colar lista" e cole vários IATAs (vírgula, espaço ou quebra de linha).
A data mínima é hoje. O horário será 15:00:00Z.
